# fsnotify 监听文件系统事件

开源库 fsnotify 是用 Go 语言编写的跨平台文件系统监听事件库，常用于文件监听，因此我们可以借助该库来实现这个功能。

```
go get -u github.com/fsnotify/fsnotify
```
