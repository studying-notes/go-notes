```shell
go help install
```

```shell
go install [build flags] [packages]
```

编译并安装源码包。

对于可执行文件，会被安装到 GOBIN （默认是 GOPATH/bin 或 $HOME/go/bin） 目录下。如果未设置 GOBIN 环境变量，则会被安装到 $GOROOT/bin 或者 $GOTOOLDIR。

无法直接指定安装路径。
