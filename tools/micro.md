## 安装

### Linux

安装在当前目录：

```shell
curl https://getmic.ro | bash
```

安装到 /usr/bin：

```shell
su - root -c "cd /usr/bin; wget -O- https://getmic.ro | GETMICRO_REGISTER=y sh"
```

```shell
cd /usr/bin
```

```shell
curl https://getmic.ro/r | sh
```

指定系统架构：

```shell
curl https://getmic.ro | GETMICRO_PLATFORM=linux64 sh
```

### Windows

```shell
choco install micro
```
